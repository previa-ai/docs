The Previa API will give developers programmatic access to prediction market data, AI confidence scores, and alert management.

<Info>
The public API is currently in development. This page will be updated with full endpoint documentation, authentication guides, and code examples when the API launches. API access will be available on **Premium** and **Institutional** plans.
</Info>

## Planned capabilities

| Capability | Description |
|---|---|
| **Market data** | Query markets across platforms with filters for category, platform, and status |
| **AI scores** | Retrieve confidence scores and breakdowns for any market |
| **Alerts** | Create, update, and manage alert rules programmatically |
| **Webhooks** | Subscribe to real-time events (score changes, alert triggers, market resolutions) |
| **Bulk export** | Download historical market data and scoring history |

## Authentication

API access will use API keys generated from your Previa account. Keys will support:

- Per-key rate limiting
- Scoped permissions (read-only, read-write)
- Usage metering and billing

## Rate limits

Rate limits will vary by plan:

| Plan | Rate limit |
|---|---|
| Premium | 100 requests/minute |
| Institutional | Custom (up to 1,000+ requests/minute) |
